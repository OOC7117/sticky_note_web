## Getting started

Open `index.html` in any modern browser. Your notes are stored in the browser, so they stay available the next time you visit.

## Development

If you prefer to run a local development server, you can use Python's built-in HTTP server:

```bash
python -m http.server 8000
```

Then navigate to [http://localhost:8000](http://localhost:8000) in your browser.
